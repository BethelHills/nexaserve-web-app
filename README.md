-NexaServe Web App
-Overview
NexaServe Web App is a portfolio-grade service booking and management web application built to model real-world workflows. It enables users to browse services, make bookings, and manage appointments, while administrators manage services, users, and bookings through a modern dashboard.

-Features
-User Features
.User registration and authentication
.Browse available services
.View service details
.Book services by date and time
.View and manage bookings
.Admin Features

-Admin authentication
.Create, update, and delete services
.View and manage all bookings
.Manage users
.Booking status tracking and reports

-Tech Stack
.Frontend: React, Cascading Style Sheets, JavaScript
.Backend: Node.js, Express
.Database: MongoDB
.Authentication: JSON Web Token
.Deployment: Vercel (frontend), Render (backend)
