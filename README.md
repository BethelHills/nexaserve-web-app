## ðŸ“Œ Project Overview

NexaServe models how a real service-based platform works end-to-end.  
Users can browse services and book appointments, while administrators manage services, bookings, and users through a dedicated admin dashboard.

This project focuses on workflow clarity, UI consistency, and professional structure, using mock data to simulate production scenarios.

## âœ¨ Features

### Public
- Browse available services
- Search, filter, and sort services
- View detailed service information
- Book services with date and time selection

### Authentication (UI)
- Login and registration screens
- Basic form validation
- Protected user flow (mocked)

### User Dashboard
- Dashboard overview
- View all personal bookings
- Track booking status (Pending, Confirmed, Completed, Cancelled)
- Cancel bookings (mock)

### Admin Dashboard
- Overview dashboard with stats
- Manage services
- Manage bookings
- Manage users (roles and status)
- Full admin navigation and layout

> Note: All data and authentication are mock implementations for demonstration purposes. Backend integration is planned.
